# SmartMed Pharmacy

SmartMed Pharmacy is a web-based pharmacy management system built using the **MERN Stack** (MongoDB, Express.js, React/HTML, Node.js).  
It helps manage medicines, sales, vendors, and customers efficiently — with a responsive dashboard for each user role.


# Customer
- Secure login and registration system  
- View available medicines  
- Place orders and view purchase history  
- Track medicine availability and expiry  

# Vendor
- Manage medicine supplies and update stock  
- Track expiry and threshold quantities  
- View order and delivery reports  

# Owner/Admin
- Dashboard with sales and profit analytics  
- Manage vendors and customers  
- Add / update / delete medicines  
- Get low-stock and expiry alerts  


# Tech Stack
 Frontend : HTML, CSS, JavaScript 
 Backend : Node.js, Express.js 
 Database : MongoDB (via Mongoose) 
 Hosting : vercel
